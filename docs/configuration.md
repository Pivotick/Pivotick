
# Configuration Options

MyLibrary supports a wide range of options:

| Option       | Type       | Default | Description                |
|--------------|------------|---------|----------------------------|
| `optionA`    | boolean    | false   | Enable feature A           |
| `optionB`    | string     | 'foo'   | Set the mode               |
| `nested`     | object     | {}      | Nested config options      |
